# 🎨 Animation Assets Download List

To make the dashboard animations work, please download the **Lottie JSON** files from the links below (or choose your own favorites) and save them into your `assets/` folder with the exact filenames listed.

| Time of Day | Filename To Save | Recommended Link |
| :--- | :--- | :--- |
| **Morning** | `sun.json` | [Sun Icon Animation](https://lottiefiles.com/free-animation/sun-mist-weather-icon-V1z5x8) |
| **Afternoon** | `cloud.json` | [Partly Cloudy Animation](https://lottiefiles.com/free-animation/partly-cloudy-day-weather-icon-A2b3c4) |
| **Evening** | `sunset.json` | [Sunset Animation](https://lottiefiles.com/free-animation/sunset-weather-icon-D5e6f7) |
| **Night** | `moon.json` | [Moon & Stars Animation](https://lottiefiles.com/free-animation/night-weather-icon-G8h9i0) |

### 📥 How to Install:
1. Click a link above (or search for "Sun", "Moon", etc. on [LottieFiles.com](https://lottiefiles.com/)).
2. Click the **Download** button.
3. Select **Lottie JSON** format.
4. Rename the downloaded file to the **Filename To Save** (e.g., `sun.json`).
5. Move the file into your project's `assets/` folder, replacing the placeholder file.
